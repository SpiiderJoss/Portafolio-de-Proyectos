Proyecto FAS MyTEAPony

2023-Presente
MyteaPony

aplicacion movil con API REST web

Junto a mi equipo de desarrollo creamos MyTEAPony, una aplicación para apoyar en las terapias a los niños con TEA, para mejorar sus habilidades cognitivas y sociales por medio de actividades configuradas por los terapeutas desde la plataforma web, mi trabajo fue desarrollar la aplicación movil, consumiendo endpoints del backend, usando peticiones de HTTP (post, put, get, update), manejando json como respuesta, además de implementar patrones de diseño tales como Factory Method, levantamiento de requerimientos y asistencia a reuniones con los clientes (CRIT TELETON Michoacán), usando metodologia AGIL SCRUM.

La idea de mytea surgio por la necesidad de apoyar a las personas que lo necesitan, dentro de los miembros del equipo de desarrollo tenemos familiares con este transtorno por lo cual para nosotros tiene mucha importancia, en México 1 de cada 115 nacen con está condición.
